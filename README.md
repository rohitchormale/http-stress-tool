# hst.py v0.2
This is simple http server stress tool

## Dependencies
twisted

## Usage
1. Open hslt.py
2. Configure METHOD, URI, HEADERS, BODY, REQUEST_INTERVAL, REQUESTS_TO_SEND as per requirements.
3. Run script as
```
python hst.py
```
